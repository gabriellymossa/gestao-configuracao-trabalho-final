## Variáveis de ambiente

O projeto utiliza variáveis de ambiente definidas no arquivo .env.

As chaves utilizadas são:

- API_URL – URL base da API (ex.: http://localhost:8080).
- APP_ENV – Ambiente da aplicação (ex.: development, production).
- DB_HOST – Host do banco de dados (ex.: localhost).
- DB_USER – Usuário do banco de dados.
- DB_PASSWORD – Senha do banco de dados.
